# Scriptly ✨

### Transforme qualquer produto em conteúdo pronto para publicar.

Scriptly é uma proposta de produto desenvolvida para o **Processo Seletivo YouShop 2026**.

A ideia nasce de uma dor recorrente na Creator Economy: creators e afiliados encontram produtos para divulgar, mas frequentemente enfrentam dificuldades para transformar esses produtos em conteúdo atrativo, consistente e capaz de gerar conversões.

O Scriptly propõe uma experiência apoiada por Inteligência Artificial Generativa que transforma páginas de vendas em roteiros estruturados para redes sociais, reduzindo o tempo de criação e acelerando a jornada entre afiliação e publicação.

---

## 📌 Sobre o Projeto

Este projeto foi desenvolvido como um exercício de Product Design, UX/UI Design e prototipação Front-End.

O objetivo não foi construir uma IA funcional, mas sim demonstrar como uma solução desse tipo poderia ser integrada ao ecossistema da YouShop através de uma experiência intuitiva, escalável e centrada no creator.

O repositório contém uma representação visual de alta fidelidade da plataforma, construída para validar:

* Fluxos de navegação;
* Arquitetura de informação;
* Experiência do usuário;
* Interface visual;
* Possíveis interações do produto.

---

## 🚀 O Problema

Embora a Creator Economy ofereça milhares de oportunidades de monetização, muitos creators enfrentam um desafio invisível:

> Produzir conteúdo em escala.

Entre as principais dores estão:

* Bloqueio criativo;
* Dificuldade em criar roteiros persuasivos;
* Tempo excessivo gasto com ideação;
* Baixa frequência de publicação;
* Dificuldade em transformar visualizações em vendas.

Como consequência, creators publicam menos conteúdo, promovem menos produtos e geram menos resultados.

---

## 💡 A Solução

O Scriptly funciona como um copiloto criativo para creators.

A proposta é simples:

1. O creator seleciona um produto.
2. A plataforma analisa a página de vendas.
3. A IA gera conteúdos estruturados prontos para adaptação e publicação.

Entre os formatos previstos:

* Reels
* TikTok
* Shorts
* Stories
* Carrosséis
* Legendas Persuasivas

---

## 🎯 Funcionalidades Propostas

### Dashboard Principal

Centraliza oportunidades, métricas e conteúdos gerados.

### Geração de Conteúdo

Fluxo de criação baseado em URLs de produtos.

### Editor Inteligente

Visualização e personalização dos roteiros gerados.

### Assistente Conversacional

Interação com IA para refinar conteúdos através de comandos rápidos.

### Histórico de Conteúdos

Organização dos materiais gerados para reutilização futura.

---

## ⚠️ Sobre este Protótipo

Esta versão não possui integração com APIs de Inteligência Artificial.

Todas as respostas, métricas e conteúdos exibidos são simulados com dados mockados e foram criados exclusivamente para representar a experiência do produto.

O foco desta entrega está em:

* Product Thinking;
* UX Design;
* UI Design;
* Arquitetura da Informação;
* Prototipação Front-End.

---
